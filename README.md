Run only setup.sh

Linux / macOS
chmod +x setup.sh

PowerShell ไม่รัน .sh โดยตรง แต่ใช้ Git Bash หรือ WSL ได้
แต่ถ้าต้องการรันผ่าน Git Bash (ที่มาพร้อม Git for Windows):

bash setup.sh
